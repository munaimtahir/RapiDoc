# RapiDoc USG (MVP)

## What this project is
RapiDoc USG is a **small, practical clinic tool** to generate **Ultrasound Abdomen** PDF reports in seconds using a **fixed AlShifa PolyClinic template** (logo + header + footer + color rules).

This MVP is intentionally narrow: it is designed to support **10–15 real reports/day** with minimal typing.

## What this project is NOT (v1)
To avoid scope creep, v1 does NOT include:
- prescriptions
- other report types
- patient registry / history
- voice input
- AI parsing
- cloud sync / accounts
- pelvis module

## Goal
One screen → pick findings → **Generate PDF** → **Print**.

Target time per report: **<10 seconds**.
