# Scope Lock — Non‑negotiable (v1)

## v1 includes (Ultrasound Abdomen only)
Inputs:
- Patient name (required)
- Age (required)
- Gender (required)
- Date/Time (auto; editable optional)

Findings (preset only):
- Liver: Fatty liver grade 1/2/3; Hepatomegaly (mild/mod); CLD (yes/no)
- Kidneys: Hydronephrosis (left/right/bilateral; mild/mod/severe); Obstruction (not seen/suspected)
- Renal stone: left/right + size (mm)
- Gallstones (yes/no)
- Ascites (mild/mod/gross)

Outputs:
- PDF in fixed **AlShifa PolyClinic** template (logo/header/footer)
- Print via Android Print Manager
- Share PDF (nice-to-have)

## v1 excludes (explicit)
No pelvis, no prescriptions, no lab reports, no EMR, no accounts, no cloud, no AI text generation.
